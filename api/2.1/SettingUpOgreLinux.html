Requirements

  • CMake 3.x
  • Mercurial. We recommend TortoiseHg
  • What you do NOT need: Boost. Don't waste your time.
  • Clang >3.5 or GCC >4.0
  • QtCreator recommended (Optional).
  • Debian-based: sudo apt-get install libfreetype6-dev libfreeimage-dev libzzip-dev libxrandr-dev libxaw7-dev freeglut3-dev libgl1-mesa-dev libglu1-mesa-dev doxygen graphviz
  • Arch: pacman -S freeimage freetype2 libxaw libxrandr mesa zziplib cmake gcc
  • For HW & SW requirements, please visit http://www.ogre3d.org/developers/requirements
  • NVIDIA users: Proprietary drivers are recommended.
  • AMD users: latest Mesa from git is recommended, though its Compute Shader support is a bit flaky at the time being (doesn't produce correct output)

Downloading Ogre

Most dependencies are conveniently stored in its own repository called "ogredeps" for you to download.

IMPORTANT: Do NOT use the the "Download" feature from Bitbucket. It will not follow subrepositories.

Command line:

1 mkdir Ogre
2 hg clone https://bitbucket.org/sinbad/ogre -r v2-1
3 cd Ogre
4 hg clone https://bitbucket.org/cabalistic/ogredeps Dependencies

Building Dependencies

1 cd Ogre/Dependencies
2 mkdir build
3 cd build
4 cmake ../
5 cd build
6 make
7 make instal

Building Ogre

We'll create both a Release & Debug configuration that match the ones used in Windows. This eases portability and cross platform development.

1 cd Ogre
2 mkdir build
3 cd build
4 mkdir Debug
5 mkdir Release
6 # Build Debug
7 cd Debug
8 cmake -D OGRE_DEPENDENCIES_DIR=Dependencies/build/ogredeps -D OGRE_BUILD_SAMPLES2=1 -D OGRE_USE_BOOST=0 -D OGRE_CONFIG_THREAD_PROVIDER=0 -D OGRE_CONFIG_THREADS=0 -D CMAKE_BUILD_TYPE=Debug ../../
9 make
10 make install
11 # Build Release
12 cd ../Release
13 cmake -D OGRE_DEPENDENCIES_DIR=Dependencies/build/ogredeps -D OGRE_BUILD_SAMPLES2=1 -D OGRE_USE_BOOST=0 -D OGRE_CONFIG_THREAD_PROVIDER=0 -D OGRE_CONFIG_THREADS=0 -D CMAKE_BUILD_TYPE=Release ../../
14 make
15 make install
